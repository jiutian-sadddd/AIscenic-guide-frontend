景区智慧导游 API 文档

后端域名：ggysummer.zeabur.app | 基础路径：/ai | 数据格式：JSON

版本：v2.0 | 更新时间：2026-07-18

通用说明

• 所有接口基础路径：/ai，完整 URL：https://ggysummer.zeabur.app/ai/...

• 统一响应结构 Result<T>：{code, message, data, success}

• code=200 表示成功，code≥400 表示失败

• JWT 鉴权：除登录/注册外，所有接口需在 Header 携带 Authorization: Bearer <token>

• SSE 流式接口（/chat/stream）返回 text/event-stream，非 JSON

• TTS 接口返回 application/octet-stream（MP3 二进制流）

• 所有接口已配置 @CrossOrigin(origins = "*")，支持跨域

目录

1. 用户认证接口（AuthController） 2. 对话接口（ChatController） 3. 历史会话接口（HistoryController） 4. 用户偏好接口（PreferencesController） 5. 语音识别接口（AsrController） 6. 语音合成接口（TtsController） 7. 知识库导入接口（ImportController） 8. 知识库管理接口（RagController） 9. 管理员接口（AdminController） 10. 数据分析接口（AnalyticsController）
1. 用户认证接口 AuthController
POST/ai/auth/register

用户注册，返回 JWT token。默认角色为 USER。

参数类型必填说明
usernameString是用户名（唯一）
passwordString是密码

请求示例

POST https://ggysummer.zeabur.app/ai/auth/register Content-Type: application/json {"username": "testuser", "password": "test123"}

响应示例

{ "code": 200, "message": "注册成功", "success": true, "data": { "userId": "efac5edb-655a-4184-aba8-99ec5d24de7c", "username": "testuser", "token": "eyJhbGciOiJIUzI1NiJ9...", "role": "USER" } }
POST/ai/auth/login

用户登录，返回 JWT token。同时支持管理员登录（role=ADMIN）。

参数类型必填说明
usernameString是用户名
passwordString是密码

请求示例

POST https://ggysummer.zeabur.app/ai/auth/login Content-Type: application/json {"username": "admin", "password": "admin123"}

响应示例

{ "code": 200, "message": "登录成功", "success": true, "data": { "userId": "2c5edab9-5438-4ae7-a7fc-53d3583e9adc", "username": "admin", "token": "eyJhbGciOiJIUzI1NiJ9...", "role": "ADMIN" } }
💡 此后所有请求需在 Header 携带：Authorization: Bearer <token>
POST/ai/auth/logout

用户注销（前端清除本地 token 即可，后端无状态）。

请求示例

POST https://ggysummer.zeabur.app/ai/auth/logout Authorization: Bearer <token>

响应示例

{ "code": 200, "message": "注销成功", "success": true, "data": null }
2. 对话接口 ChatController
GET/ai/chat

普通对话接口，返回纯文本 Markdown。前端直接 innerHTML 渲染。

参数类型必填说明
messageString否用户输入文本，默认"你好"
sessionIdString否会话 ID，默认"default"
modeString否模式：normal（普通 RAG）/ deep（Agentic RAG 深度思考），默认 normal

请求示例

GET https://ggysummer.zeabur.app/ai/chat?message=灵山胜境有哪些景点&sessionId=sess_123&mode=normal Authorization: Bearer <token>

响应示例（纯文本 Markdown）

好的，灵山胜境必看的核心景点如下： ### 1. 灵山大佛 - **位置**：祥符禅寺北侧，秦履峰南侧 - **价格**：包含在大门票内（约210元/人） - **特色**：高88米，总高101.5米，耗铜725吨 💡猜你想问： 1. 这些景点怎么安排游览顺序最省力？ 2. 灵山胜境有没有推荐的半日游路线？
GET/ai/chat/stream

SSE 流式对话接口，逐块推送 AI 回复。Content-Type 为 text/event-stream。

参数类型必填说明
messageString否用户输入文本，默认"你好"
sessionIdString否会话 ID，默认"default"
modeString否模式：normal / deep，默认 normal

请求示例

GET https://ggysummer.zeabur.app/ai/chat/stream?message=推荐美食&sessionId=sess_123&mode=normal Authorization: Bearer <token>

响应示例（SSE 流）

data: 好嘞，灵山的美食那可真是"素"得很有讲究！ data: ### 1. 梵宫素斋自助 - **位置**：灵山梵宫一楼 - **价格**：50元/位 data: ...（逐块推送）
⚠️ 此接口不能用 fetch().then(r => r.json())，必须使用 EventSource 或 ReadableStream 逐块读取。
GET/ai/chat/structured

结构化对话接口，返回标准 JSON Result<ScenicResponse>，包含景点列表和摘要。

参数类型必填说明
messageString是用户输入文本
sessionIdString否会话 ID，默认"default"
modeString否模式：normal / deep，默认 normal

请求示例

GET https://ggysummer.zeabur.app/ai/chat/structured?message=推荐景点&sessionId=sess_123 Authorization: Bearer <token>

响应示例（JSON）

{ "code": 200, "message": "查询成功", "success": true, "data": { "summary": "灵山胜境是国家5A级景区，核心景点包括...", "items": [ { "id": 1, "name": "灵山大佛", "category": "景点", "description": "高88米的青铜大佛", "location": "祥符禅寺北侧", "price": 210.0, "openTime": "全天开放", "tips": "可抱佛脚祈福，登高远眺太湖风光" } ] } }

5. 语音识别接口 AsrController
POST/ai/asr

百度短语音识别。前端将录音编码为 WAV 格式后发送。

参数类型必填说明
fileMultipartFile是WAV 格式音频文件（16kHz, 16bit, 单声道）
dev_pidint否语言模型：1537(普通话) / 1737(英语) / 1637(粤语) / 1837(四川话)，默认1537
💡 前端需先调用 MediaRecorder 录音 → encodeWAV 转为 PCM → FormData 上传。

请求示例

POST https://ggysummer.zeabur.app/ai/asr Authorization: Bearer <token> Content-Type: multipart/form-data file: [WAV 二进制数据]

响应示例

{ "code": 200, "message": "识别成功", "success": true, "data": "灵山胜境有哪些必看的景点" }
6. 语音合成接口 TtsController
GET/ai/tts

百度短文本在线合成，返回 MP3 二进制流。

参数类型必填说明
textString是要合成的文本
voiceint否音色编号，默认3（度小美）
speedint否语速 0-15，默认5
pitchint否音调 0-15，默认5
volumeint否音量 0-15，默认5

请求示例

GET https://ggysummer.zeabur.app/ai/tts?text=欢迎来到灵山胜境&voice=3 Authorization: Bearer <token>

响应示例

Content-Type: application/octet-stream Content-Disposition: attachment; filename=tts.mp3 [MP3 二进制数据]
⚠️ 此接口返回 MP3 二进制流，前端应使用 blob 方式处理：
const blob = await response.blob();
const url = URL.createObjectURL(blob);
new Audio(url).play();
7. 知识库导入接口 ImportController
POST/ai/import

上传知识库文件（Excel/Word），后端自动解析、切分、向量化后存入 Qdrant。

参数类型必填说明
fileMultipartFile是知识库文件（.xlsx / .docx），表单上传

请求示例（JavaScript）

const formData = new FormData(); formData.append('file', fileInput.files[0]); fetch('https://ggysummer.zeabur.app/ai/import', { method: 'POST', headers: { 'Authorization': 'Bearer <token>' }, body: formData }).then(r => r.json());

响应示例（成功）

{ "code": 200, "message": "文件导入成功: 灵山胜境数据集.xlsx", "success": true, "data": "灵山胜境数据集.xlsx" }

请求示例
